VacciLink Button Components

Reusable, consistent styling for our healthcare project

1. Primary Buttons

Used for main actions like "Start Session" or "Confirm".

<button class="btn btn-primary">Start Session</button>

2. Secondary Buttons

Used for navigational or less critical actions like "Back" or "Cancel".

<button class="btn btn-secondary">Back to Dashboard</button>

3. Small Variant

Used inside tables or compact lists.

<button class="btn btn-primary btn-sm">View Details</button>

4. With Icons

Buttons can include SVG icons for better visual cues.
